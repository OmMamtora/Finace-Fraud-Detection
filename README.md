# 💳 Finance Fraud Detection & Risk Visualization

## 📌 Project Overview

This project focuses on detecting fraudulent financial transactions using **Machine Learning, Big Data techniques, and Data Visualization**.

It is designed to handle **large-scale financial data (millions of records)** and provide actionable insights through an interactive system.

### 🔗 Key Components:

* 🧠 Machine Learning for fraud detection
* ⚡ Big Data handling for large datasets
* 📊 Data Visualization for risk analysis
* ⚛️ React for interactive frontend
* 🍃 MongoDB for scalable data storage

---

## 🚀 Features

* 🔍 Fraud Detection using ML algorithms
* 📊 Interactive Risk Visualization Dashboard
* ⚡ Handles large datasets (Big Data Ready)
* 🌐 Full-stack system (React + Backend API + ML)
* 📈 Model comparison (Logistic Regression, Random Forest, XGBoost, Isolation Forest)
* 🧮 Data-driven insights for financial risk

## ⚙️ Machine Learning Models

* Logistic Regression
* Random Forest
* XGBoost
* Isolation Forest (Anomaly Detection)

### 📌 Key Focus:

* Handling **imbalanced data**
* Detecting anomalies using **unsupervised learning (Isolation Forest)**
* Feature engineering
* Model evaluation using:

  * Precision
  * Recall
  * F1-score

### 💡 Why Isolation Forest?

* Detects **rare and unusual patterns (fraud cases)**
* Works efficiently on **large datasets**
* Does not require labeled data (unsupervised learning)
